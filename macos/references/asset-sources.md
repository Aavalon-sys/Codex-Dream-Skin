# 素材来源与使用边界

## 角色与颜色

- 灰泽满官方 Bilibili 置顶动态：<https://www.bilibili.com/opus/1158783344706584596>
  - 主题使用其中“手持记录板”透明立绘的裁切缩放版本。
  - Hero 内小贴纸来自同一动态中的绿冻 chibi 图。
  - 动态明确给出应援色 `#D3D3D3` 和代表 emoji `🩶`。
- 萌娘百科灰泽满条目：<https://zh.moegirl.org.cn/灰泽满>
  - 用于交叉核对代表色及“绿冻”色 `RGB(92, 150, 142)`，即 `#5C968E`。
